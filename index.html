Excel Stopping Distance Curve Fit

Interactive browser version of an Excel measurement-analysis worksheet. The tool compares vehicle stopping distances across road surfaces and estimates stopping distance using polynomial curve fitting.

Inputs

The primary assignment analysis focuses on wet road conditions. Other surfaces are included as an interactive extension.

Curve fit chart

Speed (km/h) Reaction distance (m) Braking distance (m) Stopping distance (m)

Formula

Stopping distance is calculated as reaction distance + braking distance.

Curve fitting

The trendline is a second-degree polynomial fitted to measurement points for the selected road surface.

Extrapolation

The 150 km/h result is an extrapolated estimate outside the original 30–120 km/h measurement range.
